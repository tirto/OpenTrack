FTP Download Help

Virus Protection For PCs

 1. Obtain FTP and UNZIP software

  • If you already have ftp and unzip software, please go to step 2.
  • Otherwise you will need to go the the website http://cws.internet.com and download and install both applications.

 2.Use FTP to Connect To Server

  • You can obtain the COE licensed version by using your ftp software(obtained in step 1. above) to connect to 130.65.154.41
  • Login as "anonymous" and use your e-mail address as the password.

 3. Download and Install Virus Software

  • Select your platform and your operating system from the folders listed.
  • For Windows 9x, download the file "98ixxx.zip", where xxx is the current version number.
  • For Windows NT, download the file "vntixxx.zip", where xxx is the current version number.

  • After downloading, unzip the file and install the software.
    (NOTE: If you want to put the files on diskettes to take home, download the files "98xxxl1.zip"thru "98xxxl7.zip" (Windows 9x) or "vntxxxl1.zip" thru "vntxxxl7.zip" (Windows NT), and copy each file to a diskette. Once you are at the target PC, copy all *.zip files to your hard drive, unzip the first one, and run the "install" program.).

 4. Use Current Virus Definitions! Update Frequently!

  • Now that you have version 4.x of Mcafee Viruscan, you can download the xxxxupdt.exe, where xxxx is the latest definitions version (self installing). The file is located in the /datfiles folder.
  • Alternatively, instead of the dat file download, you can instruct the software to "update" your virus definitions automatically with the Mcafee Scheduler application (Windows 9x) or the Viruscan Console (Windows NT). Choose "Update Now". You can further instruct the scheduler to update your virus definitions at a time interval you select. Once a week is recommended, as McAfee posts new updates frequently. Your computer must be ON for the updates to occur!

Virus Protection for Macs

 1. Obtain Fetch and Stuffit Expander Software

  • If you already have Fetch and Stuffit Expander, please proceed to step 2. below.
  • If you don't have Fetch or Stuffit Expander, please call Juanita Frick (x 4-3979). You won't be able to unstuff downloaded files until you obtain Stuffit Expander.
  • If you have Netscape, Stuffit Expander may be in in the Netscape folder.
  • You can also use the "Find" command to look for "Stuffit" (and "Fetch").

 2. Use Fetch To Connect To FTP Server

  • You can obtain the COE licensed version by using Fetch to connect to 130.65.154.41
  • Login as "anonymous" and use your e-mail address as the password.

 3. Download And Install Virus Software

  • Select Mac - Virus - McAfee - and download the file "virexx.hqx", where x is the current version number.
  • After downloading, unstuff the file and install the software.

 4. Use Current Virus Definitions! Update Frequently!

  • If you have version 6.x of Mcafee Virex, you can download the latest virus definitions file from Mac -Virus - McAfee -Datfile.
  • Unstuff this file and install it.

Virus Protection for Unix (including Linux)

 1. Use FTP To Connect To Server

  • Use ftp to connect to 130.65.154.41
  • Login as "anonymous" and use your e-mail address as the password.

 2. Download and Install Virus Software

  • Select the Pc - Virus - Unix folder and select your platform.
  • After downloading, expand and un-tar the file and install the software.

 3. Use Current Virus Definitions! Update Frequently!

  • Go to Pc - Virus - Datfiles and download the "datxxxx.tar", where x is the current version definitions number.
  • Un-tar this file and install it.
